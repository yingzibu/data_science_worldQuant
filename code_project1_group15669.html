Problem 1¶

Question 1a¶

Given the difference between the two models, we could get

$$\mu_i = \delta z_i + \epsilon_i$$

Thus if $\epsilon_i$ satisfies the standard assumptions $E (\epsilon_i | x_i, w_i, z_i) = 0$, thus the expectation

$$E(\mu_i | x_i, w_i) = \delta E (z_i | x_i, w_i) + E (\epsilon_i | x_i, w_i) = \delta E (z_i | x_i, w_i)$$

which is generally not 0. Therefore, $\mu_i$ may be correlated with the regressors.

Question 1b¶

Given $x \perp w$ and for variables $\tilde{x}_i = x_i - \bar{x}, \tilde{w}_i = w_i - \bar{w}, \tilde{y}_i = y_i - \bar{y}$, we have

$$\sum_i \tilde{x}_i \tilde{w}_i = 0$$

Using the model (2), which means $y_i = \alpha + \beta x_i + \gamma w_i + \mu_i$, to minimize $\sum_i (\tilde{y}_i - \beta \tilde{x}_i - \gamma \tilde{w}_i)^2$, first derivative for $\beta$

$$ \begin{aligned} \frac{\partial}{\partial \beta} \sum_i (\tilde{y}_i - \beta \tilde{x}_i - \gamma \tilde{w}_i) \tilde{x}_i & = 0 \\ \sum_i \tilde{x}_i \tilde{y}_i - \hat{\beta} \sum_i \tilde{x}_i^2 - \hat{\gamma} \sum_i \tilde{x}_i \tilde{w}_i & = 0, \text{given} \sum_i \tilde{x}_i \tilde{w}_i = 0, \text{thus} \\ \sum_i \tilde{x}_i \tilde{y}_i - \hat{\beta} \sum_i \tilde{x}_i^2 & = 0 \\ \hat{\beta} = \frac{\sum_i \tilde{x}_i \tilde{y}_i}{\sum_i \tilde{x}_i^2} = \frac{\sum_i (x_i - \bar{x}) (y_i - \bar{y}) }{\sum_i (x_i - \bar{x})^2 } \end{aligned} $$

Similarly for first derivative to $\gamma$ we have

$$\hat{\gamma} = \frac{\sum_i \tilde{w}_i \tilde{y}_i }{\sum_i \tilde{w}_i^2 } = \frac{ \sum_i (w_i - \bar{w}) (y_i - \bar{y}) }{\sum_i (w_i - \bar{w})^2 } $$

Question 1c¶

the true model is model 1, after demeaning we have

$$\tilde{y}_i = \beta \tilde{x}_i + \gamma \tilde{w}_i + \delta \tilde {z}_i + \tilde{\epsilon}_i $$

Given that from model 2

$$\hat{\gamma} = \frac{\sum_i \tilde{w}_i \tilde{y}_i }{\sum_i \tilde{w}_i^2 } = \frac{ \sum_i \tilde{w}_i (\beta \tilde{x}_i + \gamma \tilde{w}_i + \delta \tilde {z}_i + \tilde{\epsilon}_i ) }{\sum_i \tilde{w}^2 } = \frac{\beta \sum_i \tilde{w}_i \tilde{x}_i + \gamma \sum_i \tilde{w}_i^2 + \delta \sum_i \tilde{w}_i \tilde{z}_i + \sum \tilde{w}_i \tilde{\epsilon}_i }{\sum_i \tilde{w}^2 } $$

and given $x \perp w$, which means $\sum_i \tilde{w}_i \tilde{x}_i = 0$, thus

$$\hat{\gamma} = \frac{\beta \sum_i \tilde{w}_i \tilde{x}_i + \gamma \sum_i \tilde{w}_i^2 + \delta \sum_i \tilde{w}_i \tilde{z}_i + \sum \tilde{w}_i \tilde{\epsilon}_i }{\sum_i \tilde{w}^2 } = \gamma + \delta \frac{\sum_i \tilde{w}_i \tilde{z}_i }{\sum_i \tilde{w}^2 } + \frac{\sum_i \tilde{w}_i \tilde{\epsilon}_i}{\sum_i \tilde{w}^2 } $$

If the error terms $\epsilon$ satisfies the standard assumptions $E(\epsilon_i | x_i, w_i, z_i) = 0$, then

$$E(\frac{\sum_i \tilde{w}_i \tilde{\epsilon}_i}{ \sum_i \tilde{w}_i^2} | x, w, z) = 0$$

then

$$E(\hat{\gamma} | x, w, z) = \gamma + \delta \frac{\sum_i \tilde{w}_i \tilde{z}_i }{\sum_i \tilde{w}^2 }$$

thus the bias is

$$\delta \frac{\sum_i \tilde{w}_i \tilde{z}_i }{\sum_i \tilde{w}^2 } = \delta \frac{\text{Cov}(w,z)}{\text{Var}(w)} $$

Question 1d¶

The difference between model 1 and model 2 is the error term

$$\mu_i = \delta z_i + \epsilon_i $$

thus if the two models are the same, variable $z_i$ should be part of the error term instead of influencing independently the prediction.

Let $\mathbf{X} = [1, x, w]$ and $\mathbf{b} = [\alpha, \beta, \gamma]^\top$, model 1 is

$$\mathbf{y} = \mathbf{X}\mathbf{b} + \delta z + \epsilon$$

while model 2 is

$$\mathbf{y} = \mathbf{X}\mathbf{b} + \mu$$

For model 2, the OLS estimator is

$$\hat{\mathbf{b}} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y}$$

and substitute the model 1 in the equation above to get

$$\hat{\mathbf{b}} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top (\mathbf{X}\mathbf{b} + \delta z + \epsilon) = \mathbf{b} + \delta (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top z + (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \epsilon$$

Thus, to make model 1 and model 2 the same, we have to let

$$\delta (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top z = 0$$

Several options:

  1. $\delta = 0$ will make the extra term = 0

  2. $\mathbf{X}^\top z = 0$, which means $z_i$ is orthogonal to all included regressors in model 2 (i.e., the intercept, $x_i$ and $w_i$). In other words

$$\sum_i z_i = 0, \sum_i \tilde{x}_i \tilde{z}_i = 0, \sum_i \tilde{w}_i \tilde{z}_i = 0$$

where

$$\tilde{x}_i = x_i - \bar{x}, \tilde{w}_i = w_i - \bar{w}, \tilde{z}_i = z_i - \bar{z}$$

Question 1e¶

$$y_i = a + b x_i + c z_i + e_i$$

let $x_i \sim N(0, 1), e_i \sim N(0, 1)$,

$z_i$ is correlated with $x_i$, thus $\hat{b}_{model2} \rightarrow b + c \frac{\text{Cov} (X, Z)}{\text{Var} (X)}$

different funcs for relationship between $x_i$ and $z_i$

  1. $z_i = \rho x_i + \sqrt{1 - \rho^2} u_i, u_i \sim N(0, 1)$, $\hat{b}_{model2} \approx b + c \rho $

  2. $z_i = \rho x_i^2$, $cov(X, Z) = cov(X, \rho X^2) = \rho E[X^3] = 0$, thus $\hat{b}_{model2} \approx b$

  3. $z_i = \rho x_i^3$, $cov(X, Z) = \rho E[X^4] = \rho * (4-1)!! = 3 \rho$, thus $\hat{b}_{model2} \approx b + 3 \rho $

import numpy as np
import pandas as pd
import statsmodels.api as sm
import matplotlib.pyplot as plt
import seaborn as sns

a, b, c, r = 1, 2, 1, 0.5

def run_sim(n, seed = 42, zx_func=1,
            a=a, b=b, c=c, r=r):
    np.random.seed(seed)
    X = np.random.normal(0, 1, n)
    e = np.random.normal(0, 1, n)
    def get_z(zx_func=1):

        if zx_func == 1:
            Z = r * X + np.sqrt(1 - r**2) * np.random.normal(0, 1, n)
        elif zx_func == 2:
            Z = r * X**2
        elif zx_func == 3:
            Z = r * X ** 3
        return Z

    Z = get_z(zx_func)
    Y = a + b * X + c * Z + e

    # model 1: Y ~ X + Z
    X_full = sm.add_constant(pd.DataFrame({'X': X, 'Z': Z}))
    model1 = sm.OLS(Y, X_full).fit()

    # model 2: Y ~ X
    X_omit = sm.add_constant(pd.DataFrame({'X': X}))
    model2 = sm.OLS(Y, X_omit).fit()

    return model1, model2


func_dict = {1: rf'z = r x + sqrt(1 - r^2) u, r = {r}',
             2: rf'z = r x^2, r = {r}',
             3: rf'z = r x^3, r = {r}'}

for zx_func in func_dict.keys():

    results = []

When Z is correlated with X, the coefficient on X absorbs part of the effect of Z. As the sample size increases, the estimate from model 1 [correct model] converges to the true coefficient, while the model 2 converges to a biased value. Increasing sample size makes the model 2 [omitted model] estimate more stably wrong, unless X and Z are not correlated i.e. cov(X, Z) = 0.

Problem 2¶

Question 2a¶

Regression models can be sensitive to outliers since the fitted coefficients are chosen by minimizing the sum of squared residuals [e.g. shown in Problem 1]. Since residuals are squared, observations with unusually large errors will have large weight in the estimation, and these extreme observations can pull the fitted regression line toward themselves. Even a few abnormal observations can change the estimated parameters dramatically, especially when they are extreme outliers. The estimated intercept, slope coefficients, fitted values, residuals etc will be biased, which may lead to misleading conclusions.

Therefore, it is important to check the data visually and if outliers are found, they should be examined to see whether they are data errors, rare yet valid observations, or wrong data.

Question 2c¶

Already shown in Problem 1, we have

$$\hat{\beta} = \frac{\sum_{i=1}^n (x_i - \bar{x}) (y_i - \bar{y}) }{\sum_{i=1}^n (x_i - \bar{x})^2 } $$

Let $A = \sum_{i=1}^{n-1} x_i, B = \sum_{i=1}^{n-1} x_i^2$

Since the first $n-1$ observations have $y_i = 0$, $\sum_{i=1}^n x_i y_i = \sum_{i=1}^{n-1} x_i * 0 + x_n y_n = x_n y_n$

$\bar{x} = \frac{A + x_n}{n}, \bar{y} = \frac{y_n}{n}$

Thus

$$ \begin{aligned} \sum_{i=1}^n (x_i - \bar{x}) (y_i - \bar{y})&= \sum_{i=1}^n x_i y_i - n \bar{x} \bar{y} \\ & = x_n y_n - n (\frac{A + x_n}{n}) (\frac{y_n}{n}) \\ & = x_n y_n - \frac{(A + x_n) y_n}{n} \\ & = \frac{n x_n y_n - A y_n - x_n y_n }{n} \end{aligned} $$$$ \begin{aligned} \sum_{i=1}^n (x_i - \bar{x})^2 & = \sum_{i=1}^n x_i^2 - n \bar{x}^2 \\ & = B + x_n^2 - n (\frac{A + x_n}{n})^2 \\ & = B + x_n^2 - \frac{(A + x_n)^2}{n} \\ & = \frac{ nB + n x_n^2 - A^2 - 2A x_n - x_n^2 }{n} \end{aligned} $$

Thus

$$ \begin{aligned} \hat{\beta} & = \frac{n x_n y_n - A y_n - x_n y_n }{ nB + n x_n^2 - A^2 - 2A x_n - x_n^2 } \\ & = \frac{ y_n [(n-1)x_n - A] }{ (n-1)x_n^2 - 2A x_n + nB - A^2} \\ & = \frac{y_n}{x_n} \frac{ (n-1) - A / x_n }{n-1 - 2A / x_n + (nB - A^2) / (x_n^2)} \end{aligned} $$

As $x_n \rightarrow \infty$,

$$\hat{\beta} \rightarrow \frac{y_n}{x_n} \frac{ (n-1) - 0 }{n-1 - 2 * 0 + (nB - A^2) * 0} = \frac{y_n}{x_n} $$

$\square$

import itertools
y = df['Y']
predictors = ['X1', 'X2', 'X3', 'X4', 'X5']

def fit_model(cols):
    X = sm.add_constant(df[list(cols)])
    return sm.OLS(y, X).fit()

rows = []
for k in range(1, len(predictors) + 1):
    for cols in itertools.combinations(predictors, k):
        model = fit_model(cols)
        rows.append({
            'model': ' + '.join(cols),
            'num_pred': k,
            # 'R2': model.rsquared,
            'Adj_R2': model.rsquared_adj,
            'AIC': model.aic,
            'BIC': model.bic
        })

result_df = pd.DataFrame(rows)
top_adj_r2 = result_df.sort_values('Adj_R2', ascending=False).head(5)

def forward_selection(crit='AIC'):
    remaining = predictors.copy()
    selected, history = [], []
    if crit == 'Adj_R2': best_score = -np.inf
    else: best_score = np.inf

    while len(remaining) > 0:
        scores = []
        for col in remaining:
            cols = selected + [col]
            model = fit_model(cols)
            if crit == 'Adj_R2':
                score = model.rsquared_adj
            elif crit=='AIC':
                score = model.aic
            elif crit=='BIC':
                score = model.bic
            scores.append((score, col, cols, model))
        if crit == 'Adj_R2':
            best_candidate = max(scores, key=lambda x: x[0])
            improved = best_candidate[0] > best_score
        else:
            best_candidate = min(scores, key=lambda x: x[0])
            improved = best_candidate[0] < best_score
        if improved:
            best_score, best_col, best_cols, best_model = best_candidate
            selected = best_cols.copy()
            remaining.remove(best_col)

            history.append({
                'selected_model': ' + '.join(best_cols),
                'num_pred': len(best_cols),
                'R2': best_model.rsquared,
                'Adj_R2': best_model.rsquared_adj,
                'AIC': best_model.aic,
                'BIC': best_model.bic
            })
        else:
            break
    return pd.DataFrame(history), fit_model(best_cols)

Overall, all the approaches showed that the best model is Y ~ X2 + X3 + X4 + X5 and X1 should be exluded. Question 3b¶

BIC is more reliable because BIC penalizes unnecessary variables more heavily.

Let the true model

$$M_t: y_i = \alpha + \beta x_i + \epsilon_i$$

and the overfitted model

$$M_o: y_i = \alpha + \beta x_i + \gamma z_i + \mu_i$$

where in reality $\gamma = 0$ so $z_i$ is unnecessary.

Let the maximized log-likelihoods be: $l_t, l_o$, since overfitted model contains true model, we always have $l_o \geq l_t$

The likelihood-ratio statistic is $LR = 2 (l_o - l_t)$. Given $\gamma = 0$, we have $LR \sim \chi_1^2$.

$$\text{AIC} = -2l + 2k$$

The overfitted model has one extra parameter, so AIC choose the overfitted model if

$$-2l_o + 2 (k + 1) < -2 l_t + 2k$$

which is $LR > 2$.

There is always a positive probability that $\chi_1^2 > 2$. This probability does not go to 0 as $n \rightarrow \infty$. Therefore, AIC may keep selecting the unnecessary variable and is not consistent for selecting the true model.

$$\text{BIC} = -2 l + k \log n $$

BIC chooses the larger model if

$$-2 l_o + (k+1) \log n < -2 l_t + k \log n $$

which is $LR > \log n$. When the sample size n increases to infinity, $\log n \rightarrow \infty$, therefore $P(LR > \log n) \rightarrow 0$, so BIC will reject the unnecessary extra predictor with probability approaching 1.

age^2 was included because the relationship between age and wage is not likely to be purely linear. Question 4b¶

ln wage is used since wages are usually right skewed. A small number of rich people earn very high wages. Taking logs makes it less skewed and reduces the influence of extreme wage values. If often makes the regression relationship closer to linear, and helps stabilize the variance of the error term. Besides, coefficients in a log-wage model are easier to interpret as approximate percentage effects.

Question 4c¶

ln wage is used because percentage changes are usually more meaningful than absolute dollar change. E.g. $5/h increase means different for a worker earning $10/h than for a worker earning $1000/h. Using log wage allows to describe the effect of education, age, gender or degree as an approximate percentage change in wage, which is more meaningful when comparing workers with different wage levels.

Problem 5¶

Question 5a¶

$$w_t = w_{t-1} + \epsilon_t, w_0 = 0$$

thus $w_1 = w_0 + \epsilon_1 = \epsilon_1$, $w_2 = w_1 + \epsilon_2 = \epsilon_1 + \epsilon_2$ ...

thus $w_t = \epsilon_1 + \cdots + \epsilon_t = \sum_{i=1}^t \epsilon_i$,

Given $X_t = w_t + \eta_t$, thus $X_t = \sum_{i=1}^t \epsilon_i + \eta_t$

$X_t$ is not stationary as $w_t = \sum_{i=1}^t \epsilon_i$ is random walk. The variance grows with time. $\eta_t$ is only an I(0) noise component so it does not remove the random-walk component. Thus $X_t$ is nonstationary and I(1). Similarly, for Y and Z, they both contains $w_t$ and a I(0) white noise component just like X, so all X, Y and Z are nonstationary and I(1).

Question 5b¶

There are 2 cointegrated vectors. There are 3 observed series X, Y, Z yet they all have one common stochastic trend, thus the rank is 2.

Consider

$$a X_t + b Y_t + c Z_t = a (w_t + \eta_t) + b (0.5 w_t + \theta_t) + c (-w_t + \chi_t) = (a + 0.5b - c)w_t + a\eta_t + b \theta_t + c \chi_t$$

For this to be stationary,

$$a + 0.5b - c = 0$$

This gives a two-dimensional set of solutions so there are two linearly independent cointegration vectors.

One choice could be (1, -2, 0), the other could be (1, 0, 1).

For AR(2),

$$Z_t = \phi_1 Z_{t-1} + \phi_2 Z_{t-2} + \epsilon_t$$

the autocorrelations satisfy:

$$\rho_1 = \phi_1 + \phi_2 \rho_1, \rho_2 = \phi_1 \rho_1 + \phi_2$$$$\phi_1 = 0.6, \phi_2 = 0.2$$

thus

$$\rho_1 = 0.6 + 0.2 \rho_1, \rho_2 = 0.6\rho_1 + 0.2$$

thus $\rho_1 = 3/4 = 0.75$, $\rho_2 = 0.6 * 0.75 + 0.2 = 0.65$
